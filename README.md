# CS4730-Unity
This is the repo for a small Unity Game I made for my CS 4730 class at UVA.
# How to Play
- Arrow Keys to move
- Press "1" to shoot your spells!
- That's about it
# Features
- Animated Player Character
- Ability to cast spells
- EPIC boss fight
